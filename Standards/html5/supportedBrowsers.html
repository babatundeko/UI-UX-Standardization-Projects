Internet Explorer Compatibility Mode Settings

To ensure the full functionality of NHLBI applications using Internet Explorer, please use the following guidance to turn off compatibility mode.

  1. In the browser, select Tools > Compatibility View Settings to open the settings window.

  2. Uncheck the boxes "Display all websites in Compatibility View" and "Display intranet sites in Compatibility View" if they are not already unchecked.

  3. Close the settings window and restart your browser before returning to the web application.
